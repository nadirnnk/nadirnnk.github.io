Real-time Activity Detection using TinyML on an AudioMoth Device

The purpose of this project is to develop a TinyML model that can detect different activities in an environment using an AudioMoth device. The project will involve collecting a dataset of sound recordings for different activities, training a TinyML model using the dataset, and deploying the model on the AudioMoth device to detect activity in real-time.

FPGA-Based Rescue Bot with HC SR04 Ultrasonic Sensor

Modelled a Rescue Bot that can navigate its environment and detect obstacles using an FPGA board integrated with an HC SR04 ultrasonic sensor. The Rescue Bot will be controlled using VHDL coding and executed on an Altera Cyclone II FPGA board. The system is able to navigate its surroundings and detect obstacles with a minimum response time of 60 ms. This makes it a potentially valuable tool for a variety of applications, such as search and rescue, hazardous materials handling, or industrial automation.

Unmanned Fire Detection and Extinguishing System (07/2021 – 11/2022)

The objective of this project is to develop an unmanned system for detecting and extinguishing fires in order to minimize damage and save lives. The system is designed to work in both manual and automatic modes, with automatic detection and extinguishing capabilities provided by an Arduino UNO board and a flame sensor.
